What is PHP?

PHP stands for Hypertext Processor. It is a powerful tool for dynamic and interactive web pages that is widely used and downloaded for free. Scripts of this language are also executed on the server.

  • Text, HTML, CSS, JavaScript, and PHP code can all be found in PHP files.
  • The server runs PHP code, and the browser receives the output as plain HTML as result.
  • ".php" is the extension for PHP files.

Why PHP?

There are a lot of platforms which support PHP. You'll probably notice that PHP is supported by Operating Systems such as Windows, Linux, Unix, and MacOS. It also works with nearly all servers in use today (Apache, IIS, etc.) and is also compatible with a variety of databases (such as MongoDB, MySQL, PostgreSQL, and SQLite3). Php is also free-of-charge, unlike other softwares where there are subscription dues. Lastly, PHP is very simple to learn and it performs well on servers.

You can visit www.php.net to see the official PHP download page.
