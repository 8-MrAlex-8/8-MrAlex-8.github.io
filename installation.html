PHP Installation

As shown so far, PHP proves to be a very great tool for web developers to use. Now, it's time for you to try it by installing it yourself! PHP is easy to install; you just have to follow the steps below.

Note that there are 2 steps to the installation: Installing PHP itself, and installing the PHP Development Environment (ex. XAMPP).

Installing PHP

  1. Go to your browser of choice.
  2. Access https://www.php.net/downloads.php in your browser.
  3. Find "Windows Downloads" under the 'Current Stable' version of PHP.
  4. Pick the Zip file under the 'VS16 x64 Thread Safe' section.
  5. Once downloaded, right-click on the zip file and pick 'Extract All...'
  6. You may choose to rename the extracted folder into 'php-[version]'. You can do this by right-clicking on the extracted folder.
  7. Cut (Ctrl + X, or right-click >> Cut) the folder and paste it inside your Local Disk (C:) drive.
  8. Extract the file address from the address bar, open your Windows Search Bar, and type in "Edit the system environment variables".
  9. Click on Environment Variables. This will open the EV window.
  10. Click on 'Path' under 'System Variables', then click 'Edit'.
  11. Click 'New', paste the copied file directory, then click 'OK'.
  12. To verify if the installation was correct, open your Command Prompt then type "php --version". It should output the version of PHP that you've downloaded.
