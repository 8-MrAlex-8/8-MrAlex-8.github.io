PHP Installation is quite easy to follow. Just go through the following steps:

  1. 1. Go to your browser of choice.
  2. 2. Access https://www.php.net/downloads.php in your browser.
  3. 3. Find "Windows Downloads" under the 'Current Stable' version of PHP.
  4. 4. Pick the Zip file under the 'VS16 x64 Thread Safe' section.
  5. 5. Once downloaded, right-click on the zip file and pick 'Extract All...'
  6. 6. You may choose to rename the extracted folder into 'php-[version]'. You can do this by right-clicking on the extracted folder.
  7. 7. Cut (Ctrl + X, or right-click >> Cut) the folder and paste it inside your Local Disk (C:) drive.
  8. 8. Extract the file address from the address bar, open your Windows Search Bar, and type in "Edit the system environment variables".
  9. 9. Click on Environment Variables. This will open the EV window.
  10. 10. Click on 'Path' under 'System Variables', then click 'Edit'.
  11. 11. Click 'New', paste the copied file directory, then click 'OK'.
  12. 12. To verify if the installation was correct, open your Command Prompt then type "php --version". It should output the version of PHP that you've downloaded.
